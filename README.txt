Для сравнения 2х больших файлов построчно (тестировалось на файлах по 100гб), следует поместить их в папку рядом с данным скриптом и отредактировать settings.json


FirstFile - имя первого файла
LastFile - имя второго файла
OutFile - имя файла с результатами сравнения
CropFileSizeLine - кол-во строк в кропнутом файле
ThreadCount - кол-во потоков
Clean - очистить рабочие файла посде завершения работы
